# Panduan Menjalankan Aplikasi Observasi

1. Buka terminal WSL.
2. Pastikan posisi terminal ada di dalam folder project ini.
3. Jalankan perintah berikut:
   `python3 app.py`
4. Catat **PID** yang muncul di layar.
5. Biarkan terminal pertama ini tetap menyala.
6. Buka **tab terminal WSL baru**, lalu jalankan perintah `htop` atau `ps aux | grep app.py` untuk mulai melakukan analisis.
