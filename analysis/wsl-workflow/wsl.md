# Analisis WSL (OS Abstraction)

WSL (Windows Subsystem for Linux) memungkinkan Linux berjalan di atas Windows melalui lapisan abstraksi.

WSL tidak berjalan langsung di hardware, tetapi berjalan di atas Windows sebagai host. Pada WSL 2, Linux menggunakan kernel asli yang dijalankan dengan virtualisasi ringan.

Hal ini menunjukkan bahwa sistem berjalan melalui lapisan abstraksi, bukan langsung ke hardware.
